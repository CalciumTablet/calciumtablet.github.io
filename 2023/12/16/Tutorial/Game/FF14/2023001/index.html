注意！此方法已不适用于国服 6.5 及之后版本！

准备工作

提前准备好 DnSpy ，下载 dnSpy-net-win64.zip 并解压

将语言设置为中文：上方 View 👉 Language 👉 中文（中国），重启 DnSpy 即可

Dalamud.dll 修改

Dalamud.dll 位置：

  • Updater : addon\Hooks\最新版本号\Dalamud.dll

  • Launcher : Roaming\addon\Hooks\最新版本号\Dalamud.dll

将 Dalamud.dll 拖入 DnSpy ，点击上方工具栏右侧放大镜，弹出搜索栏，搜索条件改为 以上所有 + 所有文件

exe 注入器修改

注入器修改与 Dalamud.dll 修改类似

exe 文件位置：

  • Updater : Dalamud.Updater.exe

  • Launcher : app-x.x.xx-xxxx\XivLauncherCN.exe

将 exe 文件拖入 DnSpy ，点击上方工具栏右侧放大镜，弹出搜索栏，搜索条件改为 以上所有 + 所有文件

搜索 IsIntegrity ，双击搜索结果中的 IsIntegrity 跳转到目标函数

如果搜不到结果，把搜索条件从 以上所有 改成任意再改回来再次搜索就能搜到了

右键函数头并选择 编辑IL指令

在弹出窗口选中第一行(序号0)，右键并选择在 选区前添加指令 ，添加两次

修改 序号0操作码 为 ldc.i4.1 、序号1操作码 为 ret ，效果如下图，编辑后点击确定

注意事项

此方法能并非永久生效，每次卫月更新都有可能失效，重复操作一次即可再次生效

请注意，该方法目前 6.X 版本适用，持续时间未知随时可能无法使用。
