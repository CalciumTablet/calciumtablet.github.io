下载 Windows 转换脚本

首先前往 UUP dump 网站，下载脚本文件。UUP dump有很多网站，你可以在互联网上找到很多个，不用担心，他们都是可以用来下载脚本且都是很安全的，以下几个是我最常用 UUP dump 网站:

  • www.uupdump.cn 国内正常使用

  • www.uupdump.net 部分地区可能无法访问

前往 UUP dump，他们的首页会是这样的:


在这里我们可以找到近期的 Windows 更新以及之前的版本，如果想要下载最新的镜像可以从下面的 快速选项 来快速找到最新版本的镜像，当然如果你需要更早一些的镜像也可以在搜索栏进行检索。这里我们直接下载 公开发布的最新内部版本 的 x64 体系架构，直接点击后面的 x64 即可快速找到最新版本的系统。


点击后我们来到新的页面，这个页面是符合你选择的 发布类型、体系结构 以及 最新版本 三个限定条件的检索结果，这里可以看到有两条记录满足条件，其中第二个便是我们需要的系统，名称为 Windows 11,version23H2(22631.3155)amd64，我们点击它进入脚本配置页面。

第一步是选择镜像的语言，我们直接选择我们需要的并下一步即可，如果想支持更多语言可以在安装完镜像以后去系统设置里面自行添加新的语言。


第二步是选择 SKU 版本，我们不需要了解何为 SKU 版本，我们只需要对照着右边的表格来确认我们需要的 SKU 版本即可。举个例子，我要安装 Windows11 专业工作站版，对照着右边的表格可以看到所需的 SKU 版本是专业版，那我们就勾选左侧的专业版即可。确认好 SKU 版本后点击下一步。


第三步是脚本的细节内容调整，如果你很清楚自己的需求并且对其中内容有所了解可以在此处对脚本进行调整，若只想下载镜像则无需顾及任何内容直接点击创建下载包即可。


点击创建下载包后浏览器会自动下载一个很小的压缩包，大概在 10KB 左右，它就是我们在 UUP dump 配置的 Windows转换程序脚本文件。



使用脚本生成镜像文件

将下载的脚本文件压缩包解压到任意位置并确认所在盘有至少 20GB 的闲置存储空间，解压完成后我们会得到以下文件:


可以通过双击 uup_download_windows.cmd 来启动脚本，在启动脚本之前你需要知道以下内容:

  • 脚本生成镜像过程需要极长的时间，具体时间长短取决于你的硬件配置。 (硬件配置供参考: 7800x3d 需要约 20 分钟)
  • 脚本运行过程中需要全程联网，且会占用大量 CPU 资源，所以在脚本运行期间尽量不要用电脑干其他事。
  • 一定要确认有至少 20GB 闲置的储存空间，避免生成过程中由于空间不足导致的报错。
    确认完以上内容后就可以运行脚本了，之后只需等待镜像出炉即可。
 ⭐关于脚本卡在下载部分的解决办法

如果你的脚本卡在下载步骤，并且使用科学上网仍然不生效，那么你需要修改一下脚本。右键 uup_download_windows.cmd 点击编辑，在脚本头部需要修改一行代码。
在脚本的第 20 行将 :: set "all_proxy=" 改为 set "all_proxy=127.0.0.1:XXXX"，其中的 XXXX 为你的科学上网的端口，如 Clash 的 7890。以 Clash 举例如下
